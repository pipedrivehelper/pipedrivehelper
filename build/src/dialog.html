Pipedrive Helper Hotkeys

  • Alt + Shift + p : Opens Pipedrive in a new tab.
  • Alt + p : Opens Pipedrive in the current tab.
  • Alt + d : Shows Deals in Pipeline view.
  • Alt + l : Shows Deals in List view.
  • Alt + t : Shows Deals in Timeline view.
  • Alt + m : Shows Mail.
  • Alt + a : Shows Activities.
  • Alt + c : Shows People.
  • Alt + o : Shows Organizations.
  • Alt + s : Shows Statistics.
  • Alt + x : Shows Settings.
  • ` : Places the cursor in the search field.
  • Alt + h : Displays this help dialog.
